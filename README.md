# Twimber-Minimal

Minimal Desktop oriented client! Perhaps like Tweetbot?

## How to use

Download nw.js.

Edit src/cfg/default.json to reflect values, then copy to src/cfg/config.json

```
cd src/
bower install
cd src/app
bower install
cd ../..

<path/to/nw>/nw ./
```

Profit.
